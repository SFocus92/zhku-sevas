# 🔐 НАСТРОЙКА ПАРОЛЯ ДЛЯ ЗАЩИТЫ САЙТА

## ✅ Что было добавлено

- **Middleware** — проверяет авторизацию перед входом на страницы
- **Страница входа** `/login` — простой ввод пароля
- **API аутентификации** — проверяет пароль и устанавливает куку
- **Кнопка выхода** — в правом верхнем углу
- **Сессия на 30 дней** — не нужно вводить пароль каждый раз

---

## 📋 Шаг 1: Установите пароль локально

### 1.1 Откройте `.env.local`

Файл находится в корне проекта.

### 1.2 Добавьте пароль

```env
AUTH_PASSWORD="ваш_надёжный_пароль"
```

**Пример:**
```env
AUTH_PASSWORD="ZhkuSevastopol2026!"
```

---

## 📋 Шаг 2: Установите пароль на Vercel

### 2.1 Откройте Vercel Dashboard

Перейдите на https://vercel.com/dashboard

### 2.2 Выберите проект

Кликните на `zhku-sevastopol`

### 2.3 Environment Variables

1. Перейдите в **Settings** → **Environment Variables**
2. Нажмите **Add New**
3. Заполните:
   - **Name:** `AUTH_PASSWORD`
   - **Value:** ваш пароль (тот же что и локально)
   - **Environment:** ✅ Production, ✅ Preview, ✅ Development
4. Нажмите **Save**

### 2.4 Redeploy

После добавления переменной нужно сделать redeploy:

1. Перейдите в **Deployments**
2. Кликните на последний деплой
3. Нажмите **⋮** (три точки) → **Redeploy**

---

## 📋 Шаг 3: Проверка

### 3.1 Откройте сайт

Перейдите по ссылке вашего сайта (например: `https://zhku-sevastopol.vercel.app`)

### 3.2 Должно появиться окно входа

- Введите пароль
- Нажмите **Войти**
- Если пароль верный — откроется главная страница

### 3.3 Проверьте выход

- Нажмите на кнопку **🚪 Выйти** (справа вверху)
- Должно перекинуть на страницу входа

## 🛡 Безопасность

| Аспект | Защита |
|--------|--------|
| **Пароль** | Хранится в .env (не в коде) |
| **Кука** | httpOnly (не доступна из JS) |
| **Срок** | 30 дней |
| **HTTPS** | В production кука только по HTTPS |

## ⚠️ Важно!

1. **НЕ ЗАБЫВАЙТЕ ПАРОЛЬ!** 
   - Если забудете — нужно менять в Vercel и делать redeploy

2. **Одинаковый пароль везде:**
   - Локально в `.env.local`
   - На Vercel в Environment Variables

   - Используйте буквы, цифры, спецсимволы

## 🐛 Возможные проблемы

### Ошибка: "Сервер не настроен"

**Причина:** AUTH_PASSWORD не установлен

**Решение:**
1. Проверьте что переменная есть в Vercel
2. Сделайте redeploy

---

### Страница входа не появляется

**Причина:** Middleware не срабатывает

**Решение:**
1. Проверьте что файл `src/middleware.ts` существует
2. Проверьте что `AUTH_PASSWORD` установлен

---

### Выход не работает

**Причина:** Кука не удаляется

**Решение:**
1. Очистите куки в браузере
2. Попробуйте снова

---

## 📞 Поддержка

Если что-то не работает:
1. Проверьте логи в Vercel: Deployments → View Logs
2. Проверьте консоль браузера (F12)
3. Убедитесь что все файлы созданы
