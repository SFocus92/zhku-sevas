## 🔐 СЛЕДУЮЩИЙ ШАГ — УСТАНОВИ ПАРОЛЬ!

### 1. Открой `.env`
```env
AUTH_PASSWORD="ваш_пароль_здесь"
```

### 2. Замени на свой пароль
```env
AUTH_PASSWORD="ZhkuSevastopol2026!"
```

### 3. Добавь на Vercel
1. Vercel Dashboard → твой проект
2. Settings → Environment Variables
3. Add New:
   - **Name:** `AUTH_PASSWORD`
   - **Value:** твой пароль
   - **Environment:** все 3 (Production, Preview, Development)
4. Save

### 4. Сделай Redeploy
1. Deployments → последний деплой
2. ⋮ → Redeploy
