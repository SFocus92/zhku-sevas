# 🔧 ИСПРАВЛЕНИЕ: Очистка кэша и запуск в режиме инкогнито

## ✅ Что было изменено

### Проблема:
- При запуске возникали ошибки из-за кэша Next.js
- Приложение не загружалось
- Требовалась ручная очистка кэша

### Решение:
1. **Автоматическая очистка кэша** при каждом запуске
2. **Открытие браузера в режиме инкогнито** (без кэша)
3. **Полная очистка `.next` папки** перед сборкой

---

## 📁 Обновлённые файлы

### 1. `ЗАПУСТИТЬ.bat` - Основной скрипт запуска
**Что делает:**
- ✅ Удаляет папку `.next` (кэш Next.js)
- ✅ Очищает кэш npm
- ✅ Проверяет Node.js
- ✅ Проверяет зависимости
- ✅ Выполняет сборку проекта
- ✅ Копирует standalone файлы
- ✅ Запускает сервер
- ✅ Открывает браузер в режиме инкогнито

**Поддерживаемые браузеры:**
- Google Chrome (`--incognito`)
- Microsoft Edge (`-inprivate`)
- Mozilla Firefox (`-private-window`)
- Другие (открывается по умолчанию)

---

### 2. `START.bat` - Выбор режима запуска
**Добавлен новый режим:**
- **Режим 4** - "Очистить кэш и запустить" (полная очистка)

**Все режимы:**
1. Development (npm run dev) - для разработки
2. Production (standalone) - для работы
3. Quick Start (сборка + браузер) - рекомендуется
4. **Очистить кэш и запустить** - НОВИНКА!

---

### 3. `ЗАПУСК_БЕЗ_КЭША.bat` - Новый файл
**Дублирует `ЗАПУСТИТЬ.bat`** с акцентом на очистку кэша.

---

## 🚀 Как использовать

### Быстрый старт (рекомендуется):
```
ЗАПУСТИТЬ.bat
```

### С выбором режима:
```
START.bat
```
→ Выберите режим 4 для полной очистки кэша

---

## 🧹 Что очищается при запуске

| Компонент | Описание |
|-----------|----------|
| `.next/` | Папка кэша Next.js (удаляется полностью) |
| `npm cache` | Кэш пакетов npm (очищается) |
| Браузер | Открывается в режиме инкогнито (не использует кэш) |

---

## ⚠️ Важно

### Время запуска
После очистки кэша **сборка занимает больше времени** (30-60 секунд).
Это нормально - проект собирается заново.

### Не удаляйте вручную
Не нужно вручную удалять `.next` папку - скрипт сделает это сам.

### База данных
**Не удаляется!** Все ваши данные в `db/database.db` сохраняются.

---

## 🛠️ Если что-то не работает

### Белый экран после запуска:
1. Закройте окно сервера (Ctrl+C)
2. Запустите `ЗАПУСТИТЬ.bat` заново
3. Дождитесь полной сборки

### Ошибка "Node.js не найден":
- Установите Node.js с https://nodejs.org/
- Перезапустите командную строку

### Браузер не открывается в режиме инкогнито:
- Скрипт автоматически определит браузер
- Если не получилось - откроется браузер по умолчанию
- Откройте `http://localhost:3000` вручную в режиме инкогнито
