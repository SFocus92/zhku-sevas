# 🚀 ШПАРГАЛКА: Деплой на Vercel

## ⚡ Быстрый старт (5 минут)

### 1️⃣ Создать PostgreSQL базу

**Вариант A: Vercel Postgres** (проще всего)
- Зайти на [vercel.com](https://vercel.com)
- Storage → Add New → Postgres
- Скопировать `DATABASE_URL`

**Вариант B: Neon** (бесплатно)
- Зайти на [neon.tech](https://neon.tech)
- Создать проект
- Скопировать Connection String

---

### 2️⃣ Настроить локально

```bash
# 1. Скопировать шаблон
copy .env.example .env.local

# 2. Открыть .env.local и вставить DATABASE_URL

# 3. Применить схему
npm run db:push
```

---

### 3️⃣ Залить на GitHub

```bash
git init
git add .
git commit -m "Ready for Vercel"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/zhku-sevastopol.git
git push -u origin main
```

---

### 4️⃣ Подключить Vercel

1. Зайти на [vercel.com/new](https://vercel.com/new)
2. Выбрать репозиторий `zhku-sevastopol`
3. Нажать **Environment Variables**
4. Добавить `DATABASE_URL` = ваш connection string
5. Нажать **Deploy**

---

## ✅ Готово!

Сайт доступен по адресу: `https://zhku-sevastopol.vercel.app`

---

## 🔧 Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Ошибка Prisma | Проверить DATABASE_URL в Vercel |
| Нет данных | Запустить `npm run db:push` локально |
| Не грузится | Проверить логи: Deployments → View Logs |
