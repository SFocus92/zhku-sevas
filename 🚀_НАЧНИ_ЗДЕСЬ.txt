  📋 БЫСТРЫЙ СТАРТ — 4 ШАГА
═══════════════════════════════════════════════════════════════

  1️⃣  СОЗДАЙ БАЗУ ДАННЫХ (Supabase/Neon)
      👉 У вас уже подключена Supabase!
      👉 Проверьте .env с DATABASE_URL

  2️⃣  УСТАНОВИ ПАРОЛЬ
      👉 Открой .env
      👉 Найди: AUTH_PASSWORD="ваш_пароль_здесь"
      👉 Замени на: AUTH_PASSWORD="Zhku2026Sevastopol!"
      👉 Сохрани

  3️⃣  ЗАДЕПЛОЙ НА VERCEL
      👉 Запусти: ПОДГОТОВИТЬ_GIT.bat
      👉 Создай репозиторий на GitHub
      👉 Запушь: git push -u origin main
      👉 Импортируй в Vercel: vercel.com/new
      👉 Добавь переменные (см. шаг 4)

  4️⃣  ДОБАВЬ ПЕРЕМЕННЫЕ НА VERCEL
      👉 Settings → Environment Variables
      👉 Добавь:
         - DATABASE_URL (из .env)
         - AUTH_PASSWORD="твой_пароль"
      👉 Нажми Deploy
