# 🚀 ЗАПУСК ПРОЕКТА - ЖКУ СЕВАСТОПОЛЬ

## ⚡ БЫСТРЫЙ СТАРТ

### Просто запустите:
```
ЗАПУСТИТЬ.bat
```

**Что произойдёт:**
1. ✅ Очистится кэш (`.next` папка)
2. ✅ Проверятся зависимости
3. ✅ Выполнится сборка проекта
4. ✅ Скопируются standalone файлы
5. ✅ Запустится сервер
6. ✅ Откроется браузер

---

## 📱 ДОСТУП С ДРУГИХ УСТРОЙСТВ

При запуске вы увидите:
```
Локально: http://localhost:3000
В сети:   http://192.168.1.100:3000
```

**Для доступа с телефона/планшета:**
1. Откройте браузер на устройстве
2. Введите: `http://192.168.1.100:3000` (ваш IP)
3. Готово!

**Важно:**
- ✅ Устройства должны быть в **одной сети** (один Wi-Fi роутер)
- ✅ Брандмауэр Windows должен пропускать порт 3000

## 🛠️ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Белый экран / Не загружается:

1. **Очистите кэш вручную:**
   ```
   rmdir /s /q .next
   ЗАПУСТИТЬ.bat
   ```

2. **Проверьте консоль браузера (F12):**
   - Откройте DevTools (F12)
   - Посмотрите вкладку Console на ошибки

3. **Пересоберите проект:**
   ```
   npm run build
   ```

### Не подключаются другие устройства:

1. **Проверьте брандмауэр:**
   - Панель управления → Брандмауэр
   - Разрешите порт 3000 (TCP)

2. **Проверьте что устройства в одной сети**

3. **Попробуйте отключить антивирус** для проверки

## 💾 БАЗА ДАННЫХ

**Все ваши данные в `db/database.db`:**
- Тарифы
- Счета
- Показания счётчиков
- Настройки домов
- Дома и их параметры

**При переносе на другой компьютер:**
1. Скопируйте **всю папку проекта**
2. Установите Node.js на новом компьютере
3. Запустите `ЗАПУСТИТЬ.bat`

## 🎯 ВОЗМОЖНОСТИ СИСТЕМЫ

### ✅ Реализовано:
- **Газ** - расчёт по объёму помещений
- **Вода** - расчёт по счётчикам
- **Водоотведение** = сумма воды (холодная + горячая)
- **Электричество** - ступенчатый тариф + потери
- **Выгребная яма** - учёт ассенизации

### 📊 Распределение счетов:
- Электричество распределяется по **средней цене**
- Потери от ТСН добавляются к общему потреблению
- Основной дом платит остаток
