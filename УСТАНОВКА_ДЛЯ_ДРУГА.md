# 🚀 Установка для друга — ЖКУ Севастополь

Привет! Вот инструкция, как запустить систему учёта коммунальных услуг.

---

## 📋 Что нужно сделать

### Шаг 1. Установить Node.js

1. Скачай Node.js с официального сайта: **https://nodejs.org/**
2. Выбери версию **LTS** (рекомендуется, например 20.x или 22.x)
3. Запусти установщик
4. **Важно:** При установке ничего менять не нужно, просто жми "Next"
5. После установки перезагрузи компьютер (на всякий случай)

### Шаг 2. Проверить установку

Открой командную строку (Win+R → введи `cmd`) и напиши:
```
node --version
```
Должно показать версию, например: `v20.11.0`

Если показывает — всё ок! Если ошибка — переустанови Node.js.

## 📦 Что я должен тебе перекинуть

Есть **два варианта**, какой из них проще — выбирай сам.

### Вариант 1: Портативная папка (ПРОЩЕ!)

Я скидываю тебе папку **`portable`** целиком (~500 МБ).

**Что внутри:**
```
portable/
├── ЗАПУСТИТЬ.bat    ← Просто запусти этот файл!
├── standalone/      ← Файлы программы
├── db/              ← База данных (твои счета и показания)
├── prisma/          ← Настройки базы
└── .env             ← Конфигурация
```

**Как запустить:**
1. Скопируй папку `portable` в любое место (например, `C:\Programs\zhku\`)
2. Зайди в папку
3. Запусти `ЗАПУСТИТЬ.bat`
4. Откроется браузер с приложением

**Плюсы:**
- ✅ Никаких дополнительных установок
- ✅ Всё уже внутри
- ✅ Запускается в один клик

**Минусы:**
- ⚠️ Большой размер (~500 МБ)

### Вариант 2: Исходный проект (для продвинутых)

Я скидываю тебе весь проект (папку `zhku_sevastopol_MAKSIM`).

**Что делать:**
1. Скопируй папку проекта к себе (например, `C:\Projects\zhku_sevastopol_MAKSIM\`)
2. Открой командную строку в этой папке
3. Выполни команду:
   ```
   npm install
   ```
4. После установки запусти:
   ```
   npm run dev
   ```
5. Открой браузер: http://localhost:3000

**Плюсы:**
- ✅ Меньший размер при передаче
- ✅ Можно вносить изменения в код

**Минусы:**
- ⚠️ Нужно устанавливать зависимости (5-10 минут)
- ⚠️ Требуются базовые знания командной строки

## 🌐 Доступ с других устройств

Если хочешь открыть приложение с телефона или другого компьютера:

1. Узнай IP-адрес компьютера, где запущен сервер
   - При запуске он показывается в консоли
   - Например: `192.168.0.161`

2. Открой браузер на другом устройстве
3. Введи: `http://192.168.0.161:3000`

**Важно:** Устройства должны быть в одной сети (подключены к одному роутеру).

## 🔧 Если что-то пошло не так

### "Node.js не найден"

- Переустанови Node.js
- Перезагрузи компьютер
- Проверь в новой командной строке: `node --version`

### "Port 3000 already in use"

- Закрой все окна браузера с localhost:3000
- Закрой другие запущенные серверы
- В командной строке выполни:
  ```
  taskkill /F /IM node.exe
  ```
- Запусти заново

### "Prisma Client not generated"

В папке проекта выполни:
```
npx prisma generate
```

### "Ошибка базы данных"

Проверь, что папка `db` на месте и в ней есть файл `database.db`.
