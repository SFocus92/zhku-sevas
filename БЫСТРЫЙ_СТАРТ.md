# 🚀 ЖКУ Севастополь — Быстрый старт

## 📋 Что это?

Система учёта коммунальных услуг для частного дома с гостевыми домиками.

---

## ⚡ Запуск (Development)

### 1. Простой запуск

Запустите файл:
```
ЗАПУСТИТЬ.bat
```

Что делает:
- ✅ Останавливает старые процессы
- ✅ Проверяет Node.js
- ✅ Устанавливает зависимости
- ✅ Генерирует Prisma Client
- ✅ Запускает сервер
- ✅ Открывает браузер

### 2. Ручной запуск

```bash
# Установка зависимостей (первый раз)
npm install

# Запуск сервера
npm run dev
```

---

## 📦 Создание портативной версии

### Автоматически

```bash
СОЗДАТЬ_PORTABLE.bat
```

Будет создана папка `portable/` с готовой версией.

### Вручную

```bash
# Сборка проекта
npm run build

# Копирование файлов
xcopy /E /I /Y standalone portable\standalone
xcopy /E /I /Y db portable\db
xcopy /E /I /Y prisma portable\prisma
copy .env portable\.env
copy package.json portable\package.json
```

## 🔧 Команды

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск dev-сервера |
| `npm run build` | Production сборка |
| `npm start` | Запуск production |
| `npx prisma generate` | Генерация Prisma Client |
| `npx prisma db push` | Применить схему к БД |
| `npx prisma migrate dev` | Миграция БД |

---

## 🌐 Доступ из сети

После запуска доступно:
- **Локально:** http://localhost:3000
- **В сети:** http://<ВАШ_IP>:3000

Для подключения с другого устройства:
1. Откройте браузер
2. Введите IP компьютера с сервером
3. Например: `http://192.168.0.161:3000`

## 🛠 Решение проблем

### "Node.js не найден"

Установите с https://nodejs.org/ (версия LTS)

### "Port 3000 занята"

```bash
taskkill /F /IM node.exe
```

### "Prisma Client not generated"

```bash
npx prisma generate
```

### Ошибки базы данных

Проверьте путь в `.env`:
```env
DATABASE_URL="file:D:/NEW_PROGECT/zhku_sevastopol_MAKSIM/db/database.db"
```
