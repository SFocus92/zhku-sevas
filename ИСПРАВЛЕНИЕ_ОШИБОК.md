# 🛠️ ИСПРАВЛЕНИЕ ОШИБОК

## ❌ Ошибка: Can't resolve 'tailwindcss'

**Проблема:** Turbopack ищет tailwindcss в неправильной папке

### Решение 1: Очистить кэш
```cmd
rmdir /s /q .next
npm run dev
```

### Решение 2: Проверить postcss.config.mjs
Файл должен содержать:
```js
const config = {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};

export default config;
```

### Решение 3: Переустановить зависимости
```cmd
rmdir /s /q node_modules
npm install
```


Или вручную:
```cmd
rmdir /s /q .next
npm run build
node standalone\server.js
```

## ❌ Высокая нагрузка на процессор

**Проблема:** Dev-режим с Turbopack

### Решение:
Используйте **Production режим**:
```cmd
npm run build
node standalone\server.js
```

Или через START.bat → режим 2

---

## ❌ Не подключаются другие устройства

**Проблема:** Брандмауэр блокирует порт 3000

### Решение:
1. Откройте брандмауэр Windows
2. Добавьте правило для порта 3000 (TCP)
3. Или временно отключите брандмауэр для проверки

---

## 📞 Если ничего не помогает

1. Закройте все процессы Node.js в диспетчере задач
2. Удалите папки:
   ```cmd
   rmdir /s /q .next
   rmdir /s /q node_modules
   ```
3. Переустановите зависимости:
   ```cmd
   npm install
   ```
4. Запустите:
   ```cmd
   ЗАПУСТИТЬ.bat
   ```
